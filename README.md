## Starting

Git clone project
```
git clone https://github.com/developer-kirill-nefodov/weather-react-native.git

cd ./WeatherReactNative/
```

Run in dev mode
```sh
yarn install

cp .env.exemple .env

npm start
```

## Description
* Gradients: https://uigradients.com/
* Weather API: https://openweathermap.org/
* Animation: https://lottiefiles.com/
