## Starting

Git clone project
```
git clone git@github.com:developer-kirill-nefodov/WeatherReactNative.git

cd ./WeatherReactNative/
```

Run in dev mode
```sh
yarn add all

npm start
```

## Description
* Gradients: `https://uigradients.com/`
* Weather API: `https://openweathermap.org/`
* Animation: `https://lottiefiles.com/`

## Technologies:
* React
* React Native
* Expo
* Axios
